UniBridge 🌍

Connecting Future Students with Real Student Experiences

UniBridge is a modern web platform designed to help prospective international students connect with current university students across the world. It enables users to explore universities, join communities, ask questions about accommodation and visas, and get authentic advice directly from students already studying abroad.

The platform aims to simplify the decision-making process for students planning to study overseas by providing real-time insights, peer connections, and verified academic communities.

🚀 Features
1. Global Destination Explorer

Browse popular study destinations including:

United Kingdom

United States

Australia

Netherlands

Users can explore universities and student communities within each country.

2. University Directory

A searchable university listing with filters such as:

Location

Ranking

Research strength

Global recognition

Example universities included:

University of Oxford

Imperial College London

University of Cambridge

Each listing provides:

Ratings

Course availability

Student connections

3. Student Community Feed

A social feed where students share experiences about:

Accommodation

Visa processes

Scholarships

Campus life

Students can:

Post updates

Share photos

Interact with others in the community

4. Direct Student Connections

Prospective students can directly connect with verified students from their target universities to ask questions and gain insights.

5. Secure Verification System

UniBridge verifies users through:

Government ID verification

University email verification

Enrollment confirmation

This ensures that conversations happen with real and verified students.

6. Community Groups

Students can join communities based on interests such as:

UK Students

CS Majors

NYU Housing

🧠 How It Works

1️⃣ Find Your Match
Search for universities, courses, or countries.

2️⃣ Start Conversations
Connect with current students from your target university.

3️⃣ Gain Insights
Get advice about visas, housing, academics, and life abroad.

⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/yourusername/unibridge.git
2️⃣ Install dependencies
npm install
3️⃣ Configure environment variables

Create a .env.local file and add:

GEMINI_API_KEY=your_api_key_here
4️⃣ Run the development server
npm run dev

The application will start on:

http://localhost:3000
